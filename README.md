# Vue
Practical Vue.js techniques

## Development

### Scaffold
```powershell
npm create vite@latest vue -- --template vue
cd vue && npm install
```

### Run vite watcher
```bash
cd vue && npm run dev
```

### Additional packages
```bash
npm install vue-router
npm install pinia
npm install @pinia/testing --save-dev
npm i -D vitest @vitejs/plugin-vue @vue/test-utils jsdom
```
